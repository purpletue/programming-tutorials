/* learn.css */
/* CSS learning guide */
/*
   This file contains comments only. It is a study guide for CSS syntax
   and concepts, not a stylesheet to apply directly. Read the notes, then
   write your own CSS files separately.
*/
/* ---------------------------------------------------------------- */
/* 1) CSS overview */
/* ---------------------------------------------------------------- */
/* CSS controls presentation for HTML documents. It defines styles for
   layout, color, typography, spacing, and responsiveness. */
/*
   CSS rules consist of selectors and declaration blocks. Learn how
   styles cascade and how specificity determines which rules win.
*/
/* ---------------------------------------------------------------- */
/* 2) Selectors */
/* ---------------------------------------------------------------- */
/* Common selectors include element, class, ID, attribute, and pseudo-class.
   Examples:
     h1 { }
     .button { }
     #main { }
     a:hover { }
*/
/* Learn combinators, descendant selectors, and how to target the right
   elements without over-specifying.
*/
/* ---------------------------------------------------------------- */
/* 3) Properties and values */
/* ---------------------------------------------------------------- */
/* CSS properties control display, color, typography, sizing, and more.
   Examples:
     color: #333;
     background-color: white;
     font-size: 16px;
     margin: 1rem;
*/
/* Learn units like px, em, rem, %, vw, and vh. Understand when values are
   keywords, lengths, percentages, or functions.
*/
/* ---------------------------------------------------------------- */
/* 4) Box model */
/* ---------------------------------------------------------------- */
/* Every element has a box: content, padding, border, and margin.
   Learn how width and height interact with box-sizing.
*/
/* The box model is essential for layout and spacing. Practice modifying
   padding, margin, border, and box-sizing.
*/
/* ---------------------------------------------------------------- */
/* 5) Display and positioning */
/* ---------------------------------------------------------------- */
/* Control layout with display and position:
     display: block;
     display: inline-block;
     position: relative;
     position: absolute;
*/
/* Learn static, relative, absolute, fixed, and sticky positioning. Use
   display and floats carefully, and prefer modern layout methods.
*/
/* ---------------------------------------------------------------- */
/* 6) Flexbox and grid */
/* ---------------------------------------------------------------- */
/* Modern layout uses flexbox and grid.
   Example:
     display: flex;
     display: grid;
*/
/* Learn flex container properties, alignment, distribution, and grid
   template definitions for responsive layouts.
*/
/* ---------------------------------------------------------------- */
/* 7) Responsive design */
/* ---------------------------------------------------------------- */
/* Use media queries to adapt styles to different screen sizes.
   Example:
     @media (max-width: 768px) {
       .sidebar { display: none; }
     }
*/
/* Learn mobile-first design, breakpoints, and fluid layouts.
*/
/* ---------------------------------------------------------------- */
/* 8) Typography */
/* ---------------------------------------------------------------- */
/* Control fonts, line-height, letter-spacing, and text alignment.
   Example:
     font-family: Arial, sans-serif;
     line-height: 1.5;
*/
/* Learn the relationship between font settings, readability, and layout.
*/
/* ---------------------------------------------------------------- */
/* 9) Colors and backgrounds */
/* ---------------------------------------------------------------- */
/* Use color, background-color, gradients, and opacity.
   Example:
     background: linear-gradient(to right, #fff, #eee);
*/
/* Learn color contrast, accessible palettes, and how backgrounds interact
   with text and images.
*/
/* ---------------------------------------------------------------- */
/* 10) Transitions and animations */
/* ---------------------------------------------------------------- */
/* Add motion with transition and animation:
     transition: all 0.3s ease;
     animation: fadeIn 0.5s ease;
*/
/* Learn how to animate properties smoothly and avoid disruptive motion.
*/
/* ---------------------------------------------------------------- */
/* 11) Exercises to practice CSS */
/* ---------------------------------------------------------------- */
/* Write these exercises in separate .css files. Use this guide only as
   reference, and write the solutions yourself. */
/*
   Exercise 1: Basic styling
   - Style headings, paragraphs, and links.
   - Add colors and spacing.

   Exercise 2: Layout
   - Create a two-column layout with flexbox.
   - Add a responsive breakpoint.

   Exercise 3: Buttons
   - Style buttons with padding, border, and hover states.
   - Use a class selector for the buttons.

   Exercise 4: Typography
   - Set font-family, font-size, and line-height.
   - Style headings and body text separately.

   Exercise 5: Box model
   - Add padding, margin, and border to cards.
   - Use box-sizing: border-box.

   Exercise 6: Responsive design
   - Write media queries for narrow screens.
   - Adjust layout and font sizes.

   Exercise 7: Forms
   - Style input fields, labels, and submit buttons.
   - Add focus styles for accessibility.

   Exercise 8: Images and backgrounds
   - Add background color or gradient to a section.
   - Style an image with max-width and border-radius.

   Exercise 9: Grid
   - Create a responsive grid layout.
   - Place cards or items using grid-template-columns.

   Exercise 10: CSS best practices
   - Refactor selectors for clarity.
   - Remove duplicate declarations and simplify styles.
*/
/* ---------------------------------------------------------------- */
/* 12) Learning tips */
/* ---------------------------------------------------------------- */
/* Use a browser to preview your styles and developer tools to inspect
   element rules.
   Keep selectors simple and avoid overly specific rules.
   Practice writing CSS that is maintainable and accessible.
*/
/* End of CSS learning guide. */
